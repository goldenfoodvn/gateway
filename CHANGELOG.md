# Changelog

All notable changes to the Sapalens API Gateway will be documented in this file.

## [1.0.0] - 2026-02-08

### Added
- Initial release of API Gateway
- JWT authentication with JWKS support
- IP-based rate limiting
- CORS configuration
- Winston structured logging
- Health check endpoints
- Service registry and discovery
- Automatic port cleanup on startup
- Graceful shutdown handling
- Error handling middleware
- Static file serving
- Service proxy routing
- Docker support with multi-stage builds
- Comprehensive documentation

### Features
- ESM (ES Modules) support
- TypeScript with strict mode
- Modular architecture
- Centralized configuration
- Request/response logging
- Development auto-reload

### Security
- JWT token verification
- CORS protection
- Rate limiting
- Helmet.js security headers
- Non-root Docker user
